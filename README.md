# Whatsapp-Clone ❇️
WhatsDown - an messaging app like whatsapp created using React Js and Firebase.

## Tech Used:
- HTML
- CSS
- JavaScript
- React JS
- Firebase
- React Router

## Functionalities: 🎇
- Google Authentication
- Ability to create Chat Rooms
- Advanced Notification Systems

### Need to Improve: 🧑
- 1 to 1 Chat System
- Ability to upload files
